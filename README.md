HOW TO RUN

Install Node.js (LTS) from https://nodejs.org/

Clone the repository:

git clone https://github.com/{USER}/SwampStudy.git
cd SwampStudy

Install dependencies:

npm install

Start the server:

npm start

Open your browser and go to:
http://localhost:3000/
